### 🎨 Global Design Tokens (مشتركة في كل الموقع)

```css
:root {
  /* Colors */
  --primary:       #1565C0;
  --primary-dark:  #0d47a1;
  --primary-light: #1976d2;
  --navy-deep:     #0a1a3a;
  --navy-mid:      #0d2251;
  --text-dark:     #1a2035;
  --text-mid:      #4a5568;
  --text-muted:    #6b7280;
  --bg-light:      #f8f9fb;
  --border:        #e5e7eb;
  --accent-green:  #22c55e;

  /* Typography */
  --font-sans: 'Inter', 'Segoe UI', sans-serif;
  --font-display: 'Montserrat', sans-serif; /* للـ headings */

  /* Spacing */
  --section-pad: 70px 5%;
  --card-radius: 8px;
  --card-border: 1px solid #e5e7eb;

  /* Transitions */
  --transition: all 0.2s ease;
}
```

**الـ Color System:**
- Primary blue `#1565C0` — buttons, links, accents, active states
- Deep navy `#0a1a3a / #0d2251` — dark section backgrounds
- White `#ffffff` — cards, forms, light sections
- Light gray `#f8f9fb` — alternating section backgrounds
- Green `#22c55e` — dot indicators, eco tags

**الـ Typography Scale:**
- H1 Hero: 48–58px, weight 700
- H2 Sections: 26–30px, weight 700
- H3 Cards: 15–17px, weight 600
- Body: 14px, line-height 1.75
- Labels/meta: 11–13px, often uppercase + letter-spacing
